# 接口：select_terms  [返回](../README.md)
用例： [选择学期](../用例/选择学期.md)

- 功能：
    选择学期
    
- 权限：
    学生/老师：选择学期，必须先登录。    
    
- API请求地址： 
    接口基本地址/v1/api/select_terms

- 请求方式 ：
    GET

       
- 请求参数说明:        

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |user_id|用户的ID号。对应表USERS.USER_ID的值|

  
- 返回实例：

        {
                       "status": true,
                       "terms"：[
                           "2015-2016第一学期",
                           "2015-2016第二学期",
                           "2016-2017第一学期",
                           "2016-2017第二学期",
                           "2017-2018第一学期",
                           "2017-2018第一学期"
                       ]
                   }
 
    
  - 返回参数说明：    
   
    |参数名称|说明|
    |:---------:|:--------------------------------------------------------|      
    |status|bool类型，true表示正确的返回，false表示有错误|
    |terms|数组类型，元素表示学期名称|
